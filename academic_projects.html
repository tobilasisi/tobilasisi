Vehicle Telemetry

A system that provides Mgt with vehicle use and maintenance patterns

The project involves the design of a Vehicle Telemetry system that provides real-time data about vehicle fleet to support optimization and efficiency management. Features of the system include real-time overview of vehicle information, maintenance, usage & history data, and route tracking with GPS data.

Dropbox Diagnostic System

A system that reveals current dropbox utilization and maintenace conditions

The project involves the design of a system that understands dropbox usage patterns, detects over or underutilized drop boxes, and monitors maintenance history to track when maintenance is required. The system is designed to be adaptable to touch UI for tablet/mobile and mouse-driven UI for desktop.

Requirements Interaction Management System

A tool that supports the detection of pairwise interactions of software requirements

The system performs pairwise analysis, and resource-induced interaction of system requirements. Interactions between the requirements are indicated on the resulting Interaction Matrix. Interactions could be positive, partial positive, negative partial negative, or none.

Vehicle Panel Alignment Visualization System

A tool that supports users during vehicle parts alignment process on assembly lines

This project was in collaboration with the Center for Advanced Vehicular Systems (CAVS). It explores new ways to support vehicle parts alignment on vehicle assembly lines. It implements an interface that allows users to visualize measurements during the alignment of panels during vehicle alignment process.
